# 摹字 MozeScript

免費的線上中文描字帖產生器，內建唐詩、宋詞、經典散文共百餘篇，支援多種楷書與手寫字型，一鍵生成 PDF 字帖。

## 功能
- 唐詩、宋詞、經典散文精選詩詞庫
- 6 種開源楷書／手寫字型（全字庫正楷體、一明體、芫荽體、全瀨體、粉圓體、辰宇落雁體）
- 4 種格線樣式（米字格、田字格、圓框、圓框田字）
- 自訂文字輸入
- 空白格模式（純練習用）
- 電子檔與高解析印刷版 PDF 下載
- 支援匯入 GoodNotes 等筆記 App

## 技術
純前端 HTML/CSS/JavaScript，使用 jsPDF 生成 PDF，無需後端伺服器。
